List of commands:

Join/leave channel commands only works in "botbear1110" and "hotbear1110" chats.

botbear1110's chat works as a proxy chat for forsen's stream, so you will get live notifications etc. in there.

The myping command only pings you in chat, when the streamer is playing a game in your custom ping list.

In Yabbes chat, commands that aren't notify related are disabled

Join commands: Description:
bb channel join The bot joins your channel
bb channel leave The bot leaves your channel
bb channel liveemote Changes the emote for the live notifications
bb channel offlineemote Changes the emote for the offline notifications
bb channel gameemote Changes the emote for the game notifications
bb channel titleemote Changes the emote for the title notifications
Notify commands: Description:
bb notify Gives a list of available notify events
bb notify live Add's you to the live notify
bb notify title Add's you to the title notify
bb notify game Add's you to the game notify
bb remove Gives a list of available remove events
bb remove live Removes you from the live notify
bb remove title Removes you from the title notify
bb remove game Removes you from the game notify
bb gametime Responds with the amount of time, the streamer has been in the current category
Custom game pings: Description:
bb myping add *game* Add's a given game to your ping list
bb myping remove *game* Removes a given game from your ping list
bb myping remove all Removes all of the games in your ping list
bb myping list Post's a link to your ping in chat
